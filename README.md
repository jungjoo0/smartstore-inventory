# 네이버 스마트스토어 재고 관리 시스템

네이버 커머스 API를 활용한 실시간 재고 관리 웹 애플리케이션

## 주요 기능

- 📊 실시간 상품 재고 현황 조회
- 📦 옵션별 상세 재고 확인
- 🔄 자동 새로고침 기능
- 📱 반응형 웹 디자인 (모바일 지원)
- ⚠️ 재고 부족 알림

## 기술 스택

- Python 3.x
- Flask
- Naver Commerce API v2
- HTML/CSS/JavaScript

## 환경 변수 설정

`.env` 파일에 다음 정보 입력:

```
NAVER_CLIENT_ID=your_client_id
NAVER_CLIENT_SECRET=your_client_secret
```

## 로컬 실행

```bash
pip install -r requirements.txt
python app.py
```

## 배포

Render.com에서 자동 배포 가능

## UI 개선 (2025-11-22)

사용자 경험 향상 및 모바일 최적화를 위해 애플리케이션의 UI를 전면적으로 현대화했습니다.

### 주요 변경 사항

1.  **모던 디자인 시스템 도입**:
    *   **색상**: 기존의 화려한 그라데이션 배경을 제거하고, 눈이 편안한 밝은 회색(`--background-color: #f8f9fa`)을 배경으로 설정했습니다. 주요 포인트 색상으로는 차분한 파란색(`--primary-color: #007bff`)을 사용하여 전문적이고 깔끔한 느낌을 강조했습니다.
    *   **폰트**: 시스템 기본 폰트를 사용하여 어느 기기에서나 익숙하고 가독성 높은 화면을 제공합니다.

2.  **컴포넌트 스타일 개선**:
    *   **상품 카드**: 그림자와 테두리를 세련되게 다듬어 입체감을 주었으며, 내부 정보(상품명, 가격, 재고)의 시각적 계층 구조를 명확히 하여 가독성을 높였습니다.
    *   **재고 상태 시각화**: 옵션별 재고가 '부족'하거나 '품절'일 경우, 각각 경고(⚠️) 및 위험(❌) 아이콘과 색상을 사용하여 직관적으로 재고 상태를 파악할 수 있도록 개선했습니다.
    *   **버튼 및 컨트롤**: 새로고침 버튼과 로딩 인디케이터를 포함한 모든 컨트롤 요소를 깔끔하고 일관된 스타일로 재설계했습니다.

3.  **모바일 환경 최적화**:
    *   반응형 레이아웃을 유지하면서 모바일 화면에서의 여백과 폰트 크기를 미세 조정하여 가독성과 사용성을 극대화했습니다.
    *   모바일 브라우저의 주소창이나 탭 색상을 앱의 테마 색상과 일치시키는 `theme-color` 메타 태그를 추가하여 일체감 있는 사용자 경험을 제공합니다.

4.  **코드 개선**:
    *   `style.css`: 최신 CSS 변수(Custom Properties)를 기반으로 스타일을 재작성하여 유지보수성과 확장성을 높였습니다.
    *   `app.js`: 변경된 CSS 구조에 맞춰 재고 상태 클래스를 올바르게 적용하도록 스크립트를 수정했습니다.
